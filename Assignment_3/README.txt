Description:

It is a simple web-based project for denoting/creating super quadric models using different rendering techniques.

-------------/-------------/-------------/-------------/-------------

Features:

You can:

* specify Shader type (Wireframe, Phong, Gouraud);

* choose 2 models (Super-ellipsoid, Super-toroid);

* modify parameters of the chosen Super-quadric;

* rotate/zoom(in/out) the model;

* save/load your models;

* use 4 different textures for your models;
