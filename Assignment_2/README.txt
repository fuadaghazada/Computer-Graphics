Description:

It is a simple web-based animator program, with which you can create animations for the specified insect - Praying Mantis. You can view the project by opening index.html file in Source directory of the project folder.

Note: for visualizing the project you need open it on an at least 1600px width device.

-------------/-------------/-------------/-------------/-------------
Features:

* You can give a posture to the insect (Praying Mantis) by moving the sliders.
* You can save the posture (key frame) by clicking 'Save Key Frame' button.
* You can clear the listed key frames by clicking 'Clear Key Frame List' button.
* After saving a few postures, you can click 'Run' button, which will create an animation from your saved key frames.
* You can save your animation as a '.txt' file by clicking 'Save Animation' button.
* You can load an animation as a '.txt' file by clicking 'Load Animation' button.
* 'Full Screen' button is simply for getting rid of control/navigation panel for a wider view.
* For going back from Full Screen, you should simply press 'ESC' as shown in the toastr.
